I'm a full-stack software engineer with a passion for using code to understand and solve problems. I've designed, built, and solved with JavaScript/Node.js, Ruby, SQL, React, and AWS.

Most recently, I built Kuri, a Dead-Letter Queue-as-a-Service for small, distributed applications deployed on AWS infrastructure.

  • Kuri

Kuri is a Dead-Letter Queue-as-a-Service for Amazon Web Services (AWS)-Based Microservices Architectures. It provides the user with the ability to view, edit, redrive, and delete messages in their Dead-Letter Queue (DLQ).

Kuri utilizes an AWS infrastructure to construct a DLQ with extended functionality and observability for the user. Kuri can be used when building a brand new project or added to an existing project that uses a message queue.
